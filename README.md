# Personal Homepage Template (GitHub Pages)

This is a lightweight academic homepage template for **Ruijia Li**.

## Quick Start

1. Update personal links and contact in `index.html`.
2. (Optional) Replace publication/project text with your latest version.
3. Push all files to a GitHub repository.
4. Enable **GitHub Pages** from repository settings:
   - Source: `Deploy from a branch`
   - Branch: `main` (root)
5. Your page will be available at:
   - `https://<your-username>.github.io/<repo-name>/`

## Files

- `index.html`: homepage content and section structure.
- `styles.css`: visual style, responsive layout, and animations.
